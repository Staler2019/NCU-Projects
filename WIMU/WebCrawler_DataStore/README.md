# HW1: Web crawler & Data store

## Environment

- elasticsearch `7.11.2`
- openjdk `11.0.12`
- python `3.8.3`

```.cmd
pip install -r requirements.txt
```

    ```.cmd
    python main.py
    ```

- cmd

  ```.cmd
  python cmd_main.py
  ```
